# VelocityFL

VelocityFL is the **uv of Federated Learning**: Rust speed for orchestration, Python ergonomics for research.

## Why vFL

- Rust-backed orchestration with low overhead.
- Python-first API for Hugging Face, PEFT, and PyTorch users.
- Prefect-native observability for real-time round metrics.
- Built-in attack simulation for resilience testing.
